How to Test Locally

  1. Open index.html directly in your browser (double-click the file)
  2. Or use this test page to preview the app in the iframe above
  3. For mobile testing:
    • Start a local server: python3 -m http.server 8080
    • On your phone, navigate to: http://[your-computer-ip]:8080
    • Test iOS features (Dynamic Island padding, home screen icon)

Common Issues & Solutions

  • Tide data not loading: Check browser console (F12) for CORS errors. The app uses a CORS proxy which should work.
  • Bridge selector not changing spans: Check JavaScript console for errors. Make sure the page fully loaded.
  • Custom datum field not showing: Select "Custom Bridge" from dropdown - field should appear below.
  • iOS icon not showing emoji: This is expected! We replaced emoji with SVG icon for better iOS compatibility.
